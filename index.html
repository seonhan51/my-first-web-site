index.html


홈페이지를 처음 만들다 보면 아주 간단한 것에서 부터 막히는 경우가 있는데요.index 파일도 그런 것 중 하나입니다.

홈페이지를 만들어서 pc에서 테스트하고 정상적으로 작동했었는데 만든 파일을 웹호스팅 계정에 업로드하고 http://도메인 으로 접속하면 페이지가 안나오는 경우가 그런 경우입니다.

이럴 때 제일 먼저 확인해야 하는 것이 index 파일입니다.
index 파일은 홈페이지를 접속했을 때 기본으로 보여지는 파일의 파일명입니다. 확장자까지 포함하면 index.html index.htm index.php 파일들입니다.
기본으로 보여진다는 것은 http://도메인 으로 접속했을 때 index 파일이 있는지를 찾아서 맨 처음에 보여준다는 것입니다.
